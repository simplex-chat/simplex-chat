# Yhteyden muodostaminen keskustelussa

_Työn alla_

Koska sinulla ei ole tunnistetta SimpleX-alustalla, kukaan ei voi ottaa sinuun yhteyttä, ellet jaa kertakäyttöistä tai väliaikaista käyttäjäosoitetta, kuten QR-koodia tai linkkiä.

Vaikka valinnaista käyttäjäosoitetta voidaab käyttää roskapostiyhteyspyyntöjen lähettämiseen, voit muuttaa tai poistaa sen kokonaan menettämättä yhteyksiäsi.

Yksityinen yhteys - yhdistä käyttämällä kutsulinkkiä tai QR-koodia videon tai henkilökohtaisen tapaamisen kautta.

Ryhmäkeskustelu - Käyttäjillä on mahdollisuus luoda salainen ryhmä, jakaa yhteystietolinkkinsä [jonka voi poistaa myöhemmin] tai luoda kertakäyttöisen kutsulinkin.

## Keskusteluasetukset

Napauta jotakin keskusteluasi avataksesi keskusteluasetukset.
Tässä voit valita, haluatko sallia katoavat viestit, viestien poistamisen kaikilta ja ääniviestit.
